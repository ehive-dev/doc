# LEDs & Taster

## NetMode‑Taster

- Position: **unterhalb der RJ45‑Buchse**.
- Zweck: Netzwerk‑Recovery (DHCP / Default‑IP).
- Hinweis: NetMode setzt immer alle Zugangspasswörter zurück.

Details siehe *Software → NetMode (Netzwerk‑Taster)*.

## NetMode‑LED

Die NetMode‑LED signalisiert den Netzwerk‑Recovery‑Modus:

- **NetMode aktiv:** LED signalisiert/zeigt den Recovery‑Modus an.
- **NetMode inaktiv:** LED aus bzw. kein Recovery‑Signal.

Während des Tastendrucks:
- LED leuchtet dauerhaft.

## DI‑LED (EnWG §14a)

- Zeigt den Status des digitalen Eingangs (DI) an.

Hintergrund (allgemein): §14a EnWG beschreibt die netzorientierte Steuerung bestimmter Verbrauchseinrichtungen. Informationen dazu z. B. über Bundesnetzagentur/gesetzliche Grundlagen.
