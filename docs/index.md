# eHive One – Dokumentation

![eHive Logo](assets/images/ehive-logo.png){ width=220 }

Diese Dokumentation beschreibt die **Bedienung** des eHive One (Hutschienengerät-PC) und der darauf bereitgestellten Apps.

- **Gerät:** eHive One
- **Hersteller:** Loher ECS (Projekt/Plattform: eHive)
- **Zielgruppe:** Endanwender, Installateure (Betrieb/Bedienung)

## Schnellstart

1. Gerät auf Hutschiene montieren (Einbau im **geschlossenen Schaltschrank / unter Abdeckung**, IP20).
2. Versorgung anschließen (**12–30 V DC** über Klemme *oder* optional PoE – je nach Variante).
3. Netzwerk verbinden.
4. Im Browser öffnen:
   - `http://ehiveone.local/` (mDNS)
   - oder `http://<IP-ADRESSE>/` (aus dem Router/DHCP-Lease)
5. Login (Standard):
   - Benutzer: `admin`
   - Passwort: `ehive`

> Es wird **nur HTTP** bereitgestellt (kein HTTPS). Zugangsdaten nach der Inbetriebnahme ändern.

## Inhalte

- **Erste Schritte:** Inbetriebnahme, Zugriff, typische Erstkonfiguration
- **Gerät:** Anschlüsse, LEDs/Taster, technische Daten
- **Software:** Bedienung von SmartHub, WattRadar, ioManager, Tunnel Manager, NetMode
- **Wartung:** Updates, Backup/Restore, Troubleshooting
