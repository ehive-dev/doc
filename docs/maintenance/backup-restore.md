# Backup & Wiederherstellung

## Ziel

Backups sichern Konfigurationen und (je nach Anwendung) Messdaten, damit ein System nach einem Update oder Gerätetausch schnell wiederhergestellt werden kann.

## Empfehlung

- Vor größeren Änderungen/Updates ein Backup erstellen.
- Backups extern speichern (nicht nur auf dem Gerät).

## Wiederherstellung

Die konkrete Backup-/Restore‑Methode hängt vom installierten Software‑Stand (Apps/Datenhaltung) ab.

Wenn du ein Backup oder eine Wiederherstellung benötigst:

- Nutze die in den Apps vorhandenen Export-/Backup‑Funktionen (z. B. CSV‑Export in WattRadar, sofern ausreichend).
- Für vollständige Systemwiederherstellungen: Installateur/Service kontaktieren.
