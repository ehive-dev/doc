# Zugriff & URLs

## Lokal im Netzwerk

Zugriff erfolgt über das zentrale Dashboard **SmartHub** (Reverse Proxy). Einzelne App-Ports sind für Endanwender in der Regel **nicht relevant**.

Öffnen im Browser:

- `http://ehiveone.local/`
- `http://<IP-ADRESSE>/`

> Hinweis: Es wird **nur HTTP** bereitgestellt.

## Standard-Zugang

- Benutzer: `admin`
- Passwort: `ehive`

Empfehlung:
- Passwort nach Inbetriebnahme ändern.
- Remote-Zugriff nur gezielt aktivieren (siehe *Software → Tunnel Manager*).
